# Bellabeat Case Study

This analysis was made from the Bellabeat Case Study: How Can a Wellness Technology Company Play It Smart?, offered through the Google Data Analytics Certificate Capstone Project on Coursera.com


### Data Source

The data used in this case study was part of the FitBit Fitness Tracker Data, which was originally uploaded by the user Möbius on Kaggle.com and is available for public use. The data was collected from over 30 consenting
users over the period of 4/12/2016 to 5/12/2016.

Direct link below: 
https://www.kaggle.com/datasets/arashnic/fitbit?resource=download
